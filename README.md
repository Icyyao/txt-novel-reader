# txt-novel-reader
A lightweight TXT novel reader for Windows, no need extra consumption.

## 功能特点

- 📌 多书签：可添加多个书签并一键跳转
- 🔍 搜索高亮：关键词多结果高亮并跳转
- 🎯 记住上次阅读位置
- 🧘 按F11沉浸模式（无边框）
- 👻 按F11隐藏窗口
- 🌫️ 透明度滑块调整
- 🎨 自定义文字 / 背景颜色

## 下载与安装

👉 下载并运行安装程序：

**TXT小说阅读器安装程序.exe**

安装完成后桌面和开始菜单会生成快捷方式。

## 使用说明

1. 打开程序
2. 点击「文件 → 打开 TXT」
3. 选择小说文件开始阅读
